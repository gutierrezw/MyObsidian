## Decisiones técnicas

- **Causa probable desconexión IB**: `Class_websocket_IB.py` corriendo desde VS Code creaba una sesión IB competidora → gateway perdía autenticación principal. Fix: mover a AppTest (no corre automáticamente).
- **Patrón desconexión**: observado a las ~04:56 AM. Puede ser timeout del gateway IB (~6h). El nuevo banner con timestamp en el log permitirá confirmar si se repite mañana a la misma hora.
- **IB `create_session()` override**: la versión legacy bloqueaba el thread daemon con `input()`. El override loguea el banner (útil para saber cuándo re-autenticarse) pero retorna `False` inmediatamente — el `_tickle_loop` sigue reintentando.
- **BACKLOG en Obsidian vault**: elimina el ritual de cambio de rama `docs`. Claude edita directamente con Edit/Write, Obsidian Git auto-commitea cada 6hs. Fuente de verdad única.
- **40-Ideas/**: carpeta para ideas que no están listas para el BACKLOG — investigaciones técnicas y conceptos de diseño.
