# Conceptos

Ideas de diseño, patrones o enfoques que podrían aplicarse al proyecto.
No requieren investigación previa — son propuestas concretas para evaluar.

Formato sugerido para cada nota:
- **Concepto** — descripción en una oración
- **Motivación** — qué problema resuelve o qué mejora
- **Sketch de implementación** — cómo se integraría
- **Estado** — pendiente / descartado / promovido al backlog
